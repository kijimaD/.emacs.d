# .emacs.d
Emacs settings

## initialize settings

Using cask.

```shell
git clone https://github.com/cask/cask ~/.cask

cd ~/.emacs.d
sh ~/.cask/bin/cask

sudo apt-get install emacs-mozc cmigemo ripgrep
sudo apt-get install ttf-ancient-fonts # emoji
```

```
M-x (all-the-icons-install-fonts)
```
